<ul>

  <li>📷 Instant <b>QR Code Generation</b> for payments</li>
  <li>📱 Opens UPI apps (GPay, PhonePe, Paytm)</li>

  <li>⚡ Fast and lightweight frontend implementation</li>
  <li>🔐 Safe simulation (no backend required)</li>

<h2>🧠 How It Works</h2>

<ul>
  <li>1️⃣ User enters <b>UPI ID</b> and <b>Amount</b></li>
  <li>2️⃣ System generates a <b>UPI Deep Link</b></li>
  <li>3️⃣ Creates a <b>QR Code</b> for the same payment</li>
  <li>4️⃣ User can:
    <ul>
      <li>👉 Click link (mobile)</li>
      <li>👉 Scan QR (any device)</li>
    </ul>
  </li>
</ul>
