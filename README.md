<h2>🧠 How It Works</h2>
<ul>
  <li>1️⃣ User enters UPI ID</li>
  <li>2️⃣ Clicks on Pay button</li>
  <li>3️⃣ UI simulates payment process</li>
  <li>4️⃣ Shows confirmation screen</li>
</ul>
